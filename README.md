# 💸 PersonalFinanceApp

Modern kişisel finans yönetim uygulaması.  
.NET MAUI ve ASP.NET Core Web API kullanılarak geliştirilmiştir.

---

# ✨ Özellikler

- 🔐 Kullanıcı kayıt ve giriş sistemi
- 💰 Gelir / gider yönetimi
- 📊 Finansal analiz ekranı
- 📈 Grafiksel analiz sistemi
- 🧾 İşlem geçmişi görüntüleme
- ✏️ İşlem düzenleme ve silme
- 🔒 Şifre değiştirme sistemi
- 📱 Responsive modern tasarım
- 🎨 Premium fintech UI/UX deneyimi

---

# 🛠️ Kullanılan Teknolojiler

## Frontend
- .NET MAUI
- XAML
- C#

## Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

---

# 📷 Uygulama Ekranları

- Login Screen
- Register Screen
- Dashboard
- Add Transaction Page
- Statistics Page
- Graphical Analysis Page
- Change Password Page

---

# 🚀 Kurulum

## API Katmanı

```bash
cd PersonalFinance.Api
dotnet run
```

## MAUI Uygulaması

```bash
cd PersonalFinance.Maui
dotnet build
```

# 📌 Proje Amacı

Bu proje, kullanıcıların gelir ve giderlerini modern bir arayüz üzerinden yönetebilmesi, finansal durumlarını analiz edebilmesi ve harcama alışkanlıklarını grafiksel olarak görüntüleyebilmesi amacıyla geliştirilmiştir.
